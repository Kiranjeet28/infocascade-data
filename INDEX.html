🚀 Quick Commands

Run all department scrapers:

node info-data/scrapers/runAllTimetables.js

Scrape specific department (CSE):

node info-data/scrapers/cse/scrapeTimetable.js

Monitor for changes:

node info-data/scrapers/watchTimetables.js

View/edit department URLs:

cat info-data/scrapers/timetableUrls.js
